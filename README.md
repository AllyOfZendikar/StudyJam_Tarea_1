# Solucion Tarea 1 - StudyJam


Esta es mi primera tarea, haciendo la copia de una interfaz de una aplicacion movil.

## Requerimientos

  * Android Studio 2.3
  * Gradle 2.3
  * minSdkVersion 15

## Uso

Para probar este ejemplo clona este repositorio de la siguiente forma:
>
>     $ git clone https://github.com/jjcabreraparr/StudyJam_Tarea_1.git

Dentro de Android Studio:

* File --> New --> Import Project
* Seleccionas la ruta donde hiciste el `clone` del proyecto.
* Build --> Rebuild Project
* Run
